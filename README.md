## Databench 是什么？

**Databench** 是一个仿VsCode UI功能目标类似DataWorks的数据平台，

目标包含数据源管理、即席查询、文档管理、数据作业开发运维等功能

多年前开发了半个月多停滞，仅可运行shell脚本、执行sql，文件树管理、配置、及UI上的一些操作的功能。
